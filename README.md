# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data extraction and cleaning to visualization and reporting. The objective was to transform raw data into meaningful insights using Python, SQL, and Power BI.

The project includes:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL analysis using MySQL
* Interactive dashboard creation in Power BI
* Business report preparation
* Presentation development using Gamma

---

## Dataset

The dataset contains information related to the selected business domain and was used to analyze trends, patterns, and key performance indicators (KPIs).

### Dataset Features

* Multiple attributes representing business operations
* Numerical and categorical variables
* Records requiring cleaning and preprocessing

## Project Workflow

### 1. Data Loading

* Imported dataset into Python.
* Examined structure and data types.
* Identified missing values and inconsistencies.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Analyzed distributions and relationships.
* Identified trends, patterns, and outliers.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicates.
* Corrected inconsistent entries.
* Treated outliers where necessary.
* Prepared data for analysis.

### 4. SQL Analysis

* Imported cleaned dataset into MySQL.
* Executed SQL queries to:

  * Filter records
  * Aggregate data
  * Generate KPIs
  * Identify trends and insights
* Used joins, grouping, ranking, and conditional logic where applicable.

### 5. Power BI Dashboard

Developed an interactive dashboard featuring:

* Key Performance Indicators (KPIs)
* Trend Analysis
* Category-wise Performance
* Interactive Filters and Slicers
* Business Insights Visualizations

### 6. Reporting

Created a detailed analytical report summarizing:

* Objectives
* Methodology
* Findings
* Recommendations

### 7. Presentation

Prepared a professional presentation using Gamma to communicate:

* Project overview
* Analysis process
* Dashboard highlights
* Key insights
* Recommendations

## How to Run

### Python Analysis

1. Clone the repository.
2. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the Jupyter Notebook:

```bash
jupyter notebook
```

4. Run all cells in `analysis.ipynb`.

### SQL Analysis

1. Import the cleaned dataset into MySQL.
2. Open `queries.sql`.
3. Execute the queries to generate insights.

### Power BI Dashboard

1. Open `dashboard.pbix` in Power BI Desktop.
2. Refresh the dataset connection if required.
3. Explore the dashboard using available filters and slicers.
